## 4. 全局视觉规则

| 项目 | 规范 |
|---|---|
| 风格 | 白色简约、轻量毛玻璃、低饱和浅蓝 |
| 页面左右边距 | `16 px` |
| 列表项高度 | `56–64 px` |
| 主按钮高度 | `48–56 px` |
| 输入框高度 | `52–60 px` |
| 卡片圆角 | `16–20 px` |
| 大卡片圆角 | `24 px` |
| 毛玻璃区域 | TabBar、Sheet、Banner、状态卡 |
| 单屏玻璃区域上限 | 不超过 3 个大面积玻璃区域 |
| 主色 | `#1677FF` |
| 危险色 | `#D93F3F` |
| 成功色 | `#22A06B` |
